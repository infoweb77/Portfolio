Приложение Googly Puff.#

Цель - изучение технологий Grand Central Dispatch: синхронные/асинхронные функции, потоки, серийные/конкурентные очереди.

Приложение загружает фотографии из библиотеки/интернета, находит лица людей, используя Face Detection API фреймворка Сore Image и рисует им googly eyes. Функции GCD используются для оптимизации работы приложения.

×

Язык - Objective-C. Основные фреймворки: Core Graphics, Core Image, QuartzCore, UIKit.

Приложение Classic Photos.#

Цель - изучение NSOperation и NSOperationQueue.

Приложение загружает фотографии из интернета и применяет к ним фильтр Sepia фреймворка Core Image. Для ускорения работы приложения используются объекты NSOperation и очереди NSOperationQueue.

×

Язык - Swift 1.2. Основной фреймворк - Core Image.

Приложение Music Player.#

Цель - изучение процесса работы с VK iOS SDK.

Приложение получает от удаленного сервера список наиболее популярных песен, отображает его вместе с картинками, и проигрывает выбранную композицию.

×

Язык - Objective-C. Основные фреймворки: UIKit, Core Graphics, VK iOS SDK.

Приложение FlyMeThere.#

Вы указываете место, куда бы Вы хотели улететь, и приложение рассчитывает маршрут.

Маршрут включает в себя путь от текущего местоположения пользователя до ближайшего аэропорта, перелет до ближайшего аэропорта к цели путешествия и путь от аэропорта до точки назначения.

×

Язык - Objective-C. Основной фреймворк - MapKit.

Приложение iRegex.#

Цель - изучение использования регулярных выражений для поиска по тексту, а также проверки и форматирования данных, введенных пользователем.

×

Язык - Objective-C. Основные фреймворки: Core Graphics, Foundation, UIKit.

Приложение ConferencePlannerForGeeks.#

Цель - изучение работы с нативными фреймворками и приложениями Calendar, Reminder, EventKit.

Поиск конференций, планирование их посещения, добавление в календарь, напоминания про необходимость покупки билетов на мероприятие и транспорт.

×

Язык - Objective-C.
