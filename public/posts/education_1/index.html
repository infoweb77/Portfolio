Обучение

27 февраля 2016 г.

Август 2010 — Февраль 2016.#

В рамках процесса обучения, реализовал несколько десятков учебных приложений по различным темам/аспектам технологий Apple (включая такие темы, как CoreData, MapKit, PassKit, TextKit, GCD, Multitasking API и др.), а также несколько игр для iOS.

Форум игры Timezero.#

В 2013 году разработал нативное iOS-приложение для работы с форумом игры Timezero (не webview).

Цель - написание нативного приложения для работы с форумом игры (не webview).

Основной функционал - просмотр разделов и страниц, обновление статей, возможность авторизации, редатирование и публикация записей, создание новых тем. Реализовано сетевое взаимодействие, парсинг HTML, отображение контента и необходимой графики, кеширование и обновление страниц.

×

Язык - Objective-C. Основные фрэймворки: CoreText, ImageIO, Core Graphics, Core Image, UIKit, System Configuration.

Приложение Flikr Search for iPad.#

Цель - изучение поиска и загрузки фотографий с помощью Flickr API, а также возможностей UICollectionViewController.

Приложение с помощью Flickr API ищет по запросу последние загруженные фотографии на серверах Flickr, скачивает их и отображает с помощью UICollectionView. Реализованы 4 кастомные раскладки UICollectionView.

×

Язык - Objective-C. Основные фрэймворки: Core Graphics, QuartzCore, UIKit, MessageUI.

Приложение Cafe Hunter.#

Цель - изучение использования библиотек ObjC-кода в Swift-приложениях, а также работы с Facebook iOS SDK и Facebook Graph API.

Приложение использует Facebook iOS SDK и Facebook Graph API для загрузки списка кафе, расположенных поблизости от текущего местоположения пользователя. Данные отдает Facebook, кафе должно быть нем зарегистрировано..

×

Язык - Swift 1.2. Основной фрэймворк - Facebook iOS SDK.

Приложение NASA TV.#

Цель - изучение возможностей Multitasking API: фоновое обновление контента и фоновая загрузка файлов приложения.

×

Язык - Objective-C. Основные фрэймворки: Core Graphics, Security, MobileCoreServices, QuartzCore, CFNetwork, AudioToolbox, Parse, MediaPlayer, CoreData, UIKit, System Configuration, Multitasking API.

Приложение Pass Preview.#

Цель - написание приложения для загрузки и просмотра pass-билетов и купонов в симуляторе iPhone.

×

Язык - Objective-C. Основной фрэймворк - PassKit.

Приложение Textkit Magazine.#

Цель - изучение возможностей фреймворка TextKit.

Приложение загружает текстовый файл книги Alice’s adventures in wonderland, парсит, создает отображение текущей страницы в многоколоночном режиме, применяет таблицы стилей к разным фрагментам текста, организует навигацию по главам, подгружает и показывает картинки.

×

Язык - Objective-C. Основные фрэймворки: TextKit, Core Graphics, Foundation, UIKit.
