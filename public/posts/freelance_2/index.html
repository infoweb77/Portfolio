Приложение Dr.NearMe#

Приложение для поиска врачей/пациентов, записи на прием/ведения журнала приемов, систематизации контактов.

Отвечал в приложении за работу и отображение карт. Сначала работал с нативным Apple MapKit, впоследствии в команде приняли решение перейти на Google Maps. Подключил библиотеку кластеринга.

Из особенностей работы - пользователь приложения мог выбрать фильтр для отображения информации на карте - по доступности врачей (доступное время для записи / N/A в случае недоступности, специализация либо аватар доктора).

Также отвечал за верстку экранов со списками карточек докторов.
