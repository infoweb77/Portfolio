iOS-игры.#

Star Combat#

Цель - изучение программирования игр для iOS на основе фреймворка SpriteKit.

×

Язык - Objective-C. Основные фреймворки: SpriteKit, Core Graphics, Core Motion, QuartzCore, AVFoundation, UIKit.

Monkey Jump#

Цель - изучение механизмов взаимодействия игры и Apple Game Center.

На экран окончания игры показывается достигнутый результат. Также вы можете послать друзьям challenge через Game Center или опубликовать результат.

×

Язык - Objective-C. Основные фреймворки: GameKit, Core Graphics, SpriteKit, UIKit.

Hangmam#

Цель - изучение механизмов покупок в приложении (In-App Purchases) в играх для iOS.

×

Язык - Objective-C. Основные фреймворки: StoreKit, Core Graphics, QuartzCore, AVFoundation, UIKit.

Reversi#

Цель - написание игры с несложным AI на языке Swift.

Компьютер играет черными.

×

Язык - Swift 1.2. Основные фреймворки: Foundation, UIKit.

XORK!#

Цель - изучение взаимодействия JavaScript и Objective-C на примере простой текстовой игры.

Данные для игры вынесены в отдельный json-файл; логику отрабатывает JavaScript; за отображение отвечает Objective-C.

×

Языки - Objective-C, JavaScript. Основные фреймворки: Foundation, UIKit, JavaScript Core, Core Graphics.
